About the Weather App

Its so easy to receive the weather conditions in your current location. Weather web application provides detailed local forecast & weather forecast world wide, the app provides the current temperature in Celsius, sunrise and sunset time according to city time zone. The weather app also provides Wind speed,Wind Degree,Humidity. This app is made with the help of Weather by API-Ninjas which is a freely available on Rapid API website.

This is a simple weather Web Application project with HTML , CSS , JAVA SCRIPT

As this is a frontend project so only SEARCH bar will work to get the WEATHER update
